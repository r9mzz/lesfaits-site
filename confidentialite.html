Politique de confidentialité

Dernière mise à jour : 26 juin 2026

Résumé

Les Faits ne collecte aucune donnée personnelle. Aucun cookie publicitaire. Aucun tracker. Aucune donnée n'est envoyée à des serveurs tiers à votre insu.

Données stockées localement

Pour améliorer votre expérience, certaines préférences sont enregistrées dans le localStorage de votre navigateur, sur votre appareil uniquement :

  • Thème (theme) : "dark" ou "light". Stocké quand vous basculez le mode sombre.
  • Favoris (lesfaits_favs) : liste des slugs d'articles que vous avez mis en favoris.

Ces données ne quittent jamais votre navigateur. Vous pouvez les supprimer à tout moment via les paramètres de votre navigateur (Effacer les données du site).

Hébergement et logs serveur

Le site est hébergé sur GitHub Pages (GitHub, Inc.). GitHub peut conserver des logs d'accès standards (adresse IP, navigateur, pages visitées) conformément à sa propre politique de confidentialité. Les Faits n'a pas accès à ces logs.

Formulaire de contact

Les messages envoyés via notre formulaire de contact (lien mailto) sont traités par votre client de messagerie. Les Faits reçoit uniquement le contenu du message que vous choisissez d'envoyer.

Ressources externes

Certains articles contiennent des liens vers des sources externes. Ces sites tiers ont leurs propres politiques de confidentialité, sur lesquelles Les Faits n'a aucun contrôle.

Vos droits (RGPD)

Conformément au Règlement Général sur la Protection des Données (UE 2016/679), vous disposez d'un droit d'accès, de rectification et d'effacement des données vous concernant. Dans la mesure où Les Faits ne collecte aucune donnée personnelle identifiable, ces droits s'exercent principalement auprès de GitHub pour les logs serveur.

Pour toute question : lesfaits.contact@gmail.com

Modifications

Cette politique peut être mise à jour. La date de dernière modification est indiquée en haut de la page. Aucune notification individuelle n'est envoyée, puisque nous ne disposons d'aucune adresse email de nos lecteurs.
